# TelopGuard 初期設定：APIキーの取得・設定方法

TelopGuardの「AIによる文章チェック機能」を利用するためには、Googleが提供するAI（Gemini）の**APIキー**を個別に取得して設定する必要があります。
このマニュアルでは、一番簡単でエラーが少ない「Google AI Studio」を使った取得手順を説明します。

---

## 🔑 手順１：APIキーの取得（所要時間：約2分）

Googleアカウントがあれば、誰でも無料で利用枠付きのAPIキーを取得できます。

1. **Google AI Studioにアクセスする**
   以下のリンクをクリックして、APIキーの取得ページを開きます。
   👉 [Google AI Studio - Get API Key](https://aistudio.google.com/app/apikey)
   ※ご自身のGoogleアカウント（会社用など）でログインしてください。

2. **APIキーを作成する画面を開く**
   画面内にある青い **「Create API key」** ボタンをクリックします。

3. **プロジェクトを選択（または作成）する**
   - **初めて利用する場合：**「Create API key in new project」という選択肢が表示されるので、それをクリックします。
   - **すでに利用したことがある場合：**既存のプロジェクト名を選択して作成します。

4. **生成されたキーをコピーする**
   画面に `AIza` から始まる長い文字列（これがAPIキーです）が表示されます。
   右側にあるコピーボタン（📋マーク）を押して、コピーしてください。
   ※このキーはパスワードと同じくらい重要なものです。誰にも教えないように大切に扱ってください。

---

## ⚙️ 手順２：TelopGuardへの設定

キーを取得したら、あとはTelopGuardの画面に貼り付けるだけです。

1. **TelopGuardを開く**
   ブラウザでTelopGuardのチェック画面を開きます。

2. **APIキーを入力する**
   画面の右上にある「API KEY」の入力欄（Gemini API Keyと薄く書かれた場所）に、先ほどコピーした長い文字列を貼り付けます。

3. **接続テストを行う**
   入力欄の横にある **「Test」** ボタンを押します。
   画面の下部（ログ表示欄）に `✅ Connection Success!` と表示されれば、設定は完了です！

> [!NOTE]
> **APIキーの保存について**
> 入力したAPIキーは、お使いのブラウザ（今使っているパソコン内）にのみ安全に保存されます。TelopGuardの画面を閉じても保存されるため、毎回入力する必要はありません。

---

## 🆘 エラーが出た場合の対処法

> [!WARNING]
> **「サービスが無効(SERVICE_DISABLED)」といったエラーが表示される場合**
> ごくまれに、Google Cloudの管理画面（GCP）経由で手動でAPIキーを作成した場合、このエラーが発生することがあります。
> その場合、手動での復旧は少し複雑なため、上記の手順１に戻って「Google AI Studio」から新しくAPIキーを作成し直すのが一番確実で簡単です。

---

## ⏱️ 無料枠の上限（Quota）に達したときの対処法

使用量が多い日は、「1日あたりのリクエスト数の上限」に達することがあります。  
この場合、`Test` ボタンを押すと以下のようなエラーが表示されます。

```
❌ Quota exceeded / RESOURCE_EXHAUSTED
```

### 原因

TelopGuardは、AIの見落としを防ぐために**「テキストを約30行ずつに小分けにして、裏側でAIに何十回も連続してチェックを依頼する」**という設計になっています。

そのため、「ボタンを1回押しただけ」でも、AI側からは「数十回のリクエストが一気に来た」とみなされ、**無料枠の『1分間あたりの通信制限』や『1日あたりの上限』**に引っかかってしまうのが原因です。

### 対処法

**方法①：翌日まで待つ（最も簡単）**

制限は毎日、日本時間の16〜17時頃（米国太平洋時間の午前0時頃）にリセットされます。  
その後に `Test` ボタンを押し直すと、通常通り使用できます。

**方法②：別のGoogleアカウントでAPIキーを再発行する（すぐ使いたい場合）**

> [!NOTE]
> 無料枠は「Googleアカウント（プロジェクト）」単位で管理されています。**同じアカウント内でキーを何個作っても、上限枠は増えません。**
> 一時的に上限を回避するには、**仕事用や別管理用の「別のGoogleアカウント」**にログインして、新しいキーを発行する必要があります。

1. 現在とは別のGoogleアカウントに切り替える（またはログインし直す）
2. [Google AI Studio - Get API Key](https://aistudio.google.com/app/apikey) を開く
3. 新しいAPIキーを作成する（「Create API key in new project」を選択）
4. 生成されたキーをコピーして、TelopGuardのAPI KEY欄に貼り付け直す
