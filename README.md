# TelopGuard: AI Telop Correction Tool

TelopGuard（テロップガード）は、動画編集者向けのテロップ校正ツールです。
Google Gemini APIを使用して、テロップの誤字脱字、表記揺れ、ニュアンスの違和感を自動検出し、修正案を提示します。

## 主な機能
- **自動校正**: テキストファイル（.txt）をドラッグ＆ドロップするだけでAIがチェック。
- **ハイブリッド判定**:
  - **プログラム判定**: 数字の半角統一、不要な行の検出など、絶対的なルールを適用。
  - **AI判定**: 文脈に応じた誤変換（例：上手くいく vs 上手く言う）やニュアンスをチェック。
- **重複行アラート**: 前の行と同じ内容が含まれている場合、即座に警告。
- **BYOK (Bring Your Own Key)**: APIキーはブラウザに保存され、サーバーには送信されません。

## 使い方
1. **APIキーの取得**:
   - [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセスし、APIキー（無料枠あり）を取得してください。
2. **アプリの起動**:
   - 右上の「API KEY」欄にキーを入力し、必要なら「Test」で接続確認をします。
3. **チェック実行**:
   - VrewやPremiere Proから書き出したテキストファイル（.txt）をアップロード。
   - 自動的にフォーマット是正が行われます。
   - 「Run AI Check」ボタンを押すと、AIによる意味チェックが実行されます。

## 配布・デプロイ方法（他の人と共有する場合）
このアプリは静的なWebサイトとして動作します。以下のいずれかの方法で共有できます。

### A. Vercelで公開（推奨）
最も簡単に、URLで共有できる方法です。
1. [Vercel](https://vercel.com/)のアカウントを作成。
2. このプロジェクトフォルダをGitHubにアップロード、またはVercel CLIを使ってデプロイ。
3. 発行されたURLを共有するだけで、誰でも自分のAPIキーを入れて使えます。

### B. ローカルで実行
エンジニア以外の方には推奨しませんが、`.zip`等で渡す場合：
1. `Node.js` が必要です。
2. フォルダ内で `npm install` -> `npm run dev` を実行するとブラウザで開けます。
